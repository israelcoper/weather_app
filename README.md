# README

* Ruby 2.7.4

* Rails 5.1.7

* Setup
    - bundle install

* Database creation
    - rake db:create
    - rake db:migrate

* Create config file then provide the weatherapi key
    - cp config/application.yml.sample config/application.yml

* How to run the test suite
    - rspec

* How to run the app
    - rails s
